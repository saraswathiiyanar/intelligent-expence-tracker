# 💰 Intelligent Expense Tracker with Financial Advisory System

An intelligent web-based expense tracking application built using **Python Flask**, SQLite, HTML, CSS, and JavaScript. This system helps users manage daily expenses and provides smart financial advice based on spending patterns.

---

## 🚀 Features

- User Login & Logout system
- User Registration page
- Add daily expenses (Category, Amount, Date)
- View all expenses in a table format
- Delete unwanted expenses
- Total expense calculation
- Category-wise expense analysis
- Monthly expense report
- Smart financial advice based on total spending
- User profile page

---

## 🛠️ Tech Stack

- Python (Flask)
- SQLite Database
- HTML5
- CSS3
- JavaScript

---

## 📊 Functionalities

- Expense tracking with database storage
- Automatic financial advice generation
- Data grouping by category and month
- Session-based login system

---

## 📁 Project Structure

- app.py → Main Flask application
- templates/ → HTML files
- static/ → CSS & JS files
- expense.db → SQLite database
